# Instant-Hoppers

## Lightning-Fast Item Transfers

Say goodbye to hopper delays! This mod eliminates the 8-tick cooldown from vanilla hoppers, allowing instant item transfers between containers while maintaining full compatibility with existing hopper mechanics.

### Key Features
- **Instant transfers** - Items move between connected hoppers and containers immediately
- **Seamless integration** - Works with all vanilla containers and modded inventories
- **Bulk movement** - Transfers entire stacks at once rather than single items
- **Survival-friendly** - No configuration needed, just place hoppers as normal

Perfect for complex redstone contraptions and storage systems where timing matters. Works exclusively with Minecraft 1.12.2 on Forge.
