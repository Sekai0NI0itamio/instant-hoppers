# Instant-Hoppers

## What It Does

- Hoppers pull items from the block they face and push them into the block they’re attached to, moving whole stacks in a single burst.
- They also instantly suck up items that are dropped on top of them, so you don’t have to wait for them to pick up one by one.
- Hoppers can move full stacks in one go instead of only one item at a time.
- Hoppers also pull dropped items on top into themselves right away.
